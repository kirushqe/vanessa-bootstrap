Предназначена для сохранения настройки выгрузки.

Открывается с помощью кнопки Настройки из окна Интерактивное изменение выгрузки.

В списке приводятся Ранее сохраненные настройки.

Заполните Имя сохраняемой настройки. Двойным щелчком мыши можно выбрать из списка одну из ранее сохраненных настроек для перезаписи.

Нажмите Сохранить. Последняя отправка сохраняется автоматически.
